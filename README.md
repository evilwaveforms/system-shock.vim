# system-shock.vim

Dark minimalistic vim color scheme with some variations.

## Setup


With [vim-plug](https://github.com/junegunn/vim-plug)
```
Plug 'evilwaveforms/system-shock.vim'

set termguicolors
colorscheme system-shock
```
